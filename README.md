# Карта
Приложение для отображения гео-маркеров на карте. 

![Demo video](map_demo.gif)

# Используемые технологии
* TypeScript
* React
* Redux Toolkit
* MUI
* Axios
* Formik
* Mapbox GL JS
* React Map GL
* eslint

# Запуск и установка 
```
git clone git@github.com:po1inakoroleva/map.git
```
```
npm ci
```
```
npm start
```
